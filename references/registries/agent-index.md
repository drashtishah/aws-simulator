## Critical paths

The `guard-write` hook in `.claude/hooks/guard-write.ts` blocks writes to:

- File: `references/registries/path-registry.csv` (auto-generated by `scripts/extract_paths.py`)
- File: `learning/logs/activity.jsonl` (append-only log)
- File: `package-lock.json`
- Directory: `node_modules/`
- Directory: `web/test-specs/`

Edits to these paths must go through their generators or the appropriate skill flow.
