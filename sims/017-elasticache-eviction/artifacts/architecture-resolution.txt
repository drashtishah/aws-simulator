Root cause: maxmemory-policy = noeviction
  -> Redis rejects writes when memory is full
  -> Application silently falls through to database
  -> Database connection pool saturates under full load
  -> ALB TargetResponseTime climbs to 8.2 seconds

Fix: CONFIG SET maxmemory-policy allkeys-lru
     + CloudWatch alarms on BytesUsedForCache, CacheMisses
     + Scale node or enable cluster mode
