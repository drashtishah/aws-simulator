  Flow:
    1. Browser calls API Gateway to get a presigned PUT URL
    2. Lambda generates presigned URL for S3 and returns it
    3. Browser uses presigned URL to PUT file directly to S3 via CloudFront
    4. S3 receives PUT -- but browser sends OPTIONS preflight first

  Why curl/Postman work:
    - They skip step 4 entirely. No preflight. No CORS check.
    - The PUT goes through. S3 accepts it. The presigned URL is valid.
