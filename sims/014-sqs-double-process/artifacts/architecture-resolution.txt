  FAILURE MODE:
  1. Large order (20+ items) arrives in queue
  2. Lambda invocation #1 begins processing (takes 60-90s)
  3. At 30s mark, SQS visibility timeout expires
  4. Message becomes visible again
  5. Lambda invocation #2 picks up the same message
  6. Both invocations complete: double charge, double inventory decrement
