# AWS Services Catalog

Reference catalog of AWS services covered across the SAA-C03, SAP-C02, DVA-C02, and SCS-C02 certification exams. Used by the simulation system to track knowledge gaps and prioritize learning.

**Exam priority scale:** 1 = Always tested, 2 = Frequently tested, 3 = Regularly tested, 4 = Occasionally tested
**Difficulty scale:** 1 = Foundational, 2 = Associate, 3 = Developer/Security specialty, 4 = Professional
**Knowledge score:** Not started = 0, In progress = 1-3, Familiar = 4-6, Confident = 7-9, Mastered = 10
